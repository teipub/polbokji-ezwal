# config.json

```json
{
  "id": "jjuson",
  "password": "ja15909090!",
  "date": "20240116",
  "preferredAccommodations": [
    10052
  ],
  "roomName": "4인",
  "night": 2,
  "waitMilliseconds": 1000
}
```

| 키 | 설명 |
| id | ID |
| password | 비밀번호 |
| date | 예약 날짜 |
| preferredAccommodations | 선호 숙소 ID (아래 참조)|
| roomName | 예약할 방 이름 |
| night | 예약할 박 수 |
| waitMilliseconds | 예약 시도 간격 (밀리초) |
